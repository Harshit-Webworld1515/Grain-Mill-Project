# Grain Mill Website Clone

This is a simple front-end project that replicates a **Grain Mill website** using **HTML** and **CSS**.  
The project demonstrates a clean layout for showcasing grain products, services, and company information.

## Features
- Responsive website layout
- Navigation bar with links to Home, Products, About, and Contact
- Product section displaying different types of grains
- Services and About section
- Footer with contact information

## Technologies Used
- HTML5
- CSS3
- Flexbox and Grid for layout
- Hover effects and styling for a professional look
